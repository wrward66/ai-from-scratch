# Linear Regression from Scratch
In this project, I implemented Linear Regression using only NumPy, trained using gradient descent.

## Concepts Covered
- Mean Squared Error loss
- Gradient descent optimization
- Parameter updates
- Visualization of fit and loss curve

## Results
- True parameters: w=3.0, b=4.0
- Trained parameters: w=2.77, b=4.22
